# URL-параметры страниц бронирования

Страницы `/new/private` и `/new/shared` поддерживают предзаполнение формы через GET-параметры.

## Параметры

### Shared `/new/shared`

| Параметр | Тип    | Обязательный | Описание                             |
|----------|--------|:------------:|--------------------------------------|
| `date`   | string | ❌           | Дата в формате `YYYY-MM-DD`          |
| `adults` | int    | ❌           | Количество взрослых (по умолчанию 1) |
| `kids`   | int    | ❌           | Количество детей (по умолчанию 0)    |
| `tour`   | int    | ❌           | ID тура — предвыбирает лодку         |

### Private `/new/private`

| Параметр | Тип    | Обязательный | Описание                             |
|----------|--------|:------------:|--------------------------------------|
| `date`   | string | ❌           | Дата в формате `YYYY-MM-DD`          |
| `adults` | int    | ❌           | Количество взрослых (по умолчанию 1) |
| `kids`   | int    | ❌           | Количество детей (по умолчанию 0)    |
| `tour`   | int    | ❌           | ID тура — предвыбирает лодку         |
| `route`  | int    | ⚠️ если `tour` | ID маршрута — обязателен при передаче `tour` |

## Примеры

### Shared
```
https://bluuu.tours/new/shared?date=2026-07-01&adults=2&kids=1
https://bluuu.tours/new/shared?date=2026-07-01&adults=2&kids=1&tour=20
```

### Private
```
https://bluuu.tours/new/private?date=2026-07-01&adults=2&kids=1
https://bluuu.tours/new/private?date=2026-07-01&adults=2&kids=1&tour=1&route=5
```

## Поведение

- Если `date` передан — страница открывается в режиме **конкретной даты** (не «гибкая дата»), календарь позиционируется на нужном месяце.
- Если `date` не передан — стандартное состояние («выбрать позже»).
- `tour` соответствует `id` из `/api/chatbot/boats/private` или `/api/chatbot/boats/shared`.
- `route` соответствует `id` из массива `routes` в `/api/chatbot/boats/private`.
- Для private: если передан `tour` без `route` — лодка предвыбрана, маршрут не выбран.
